VIPER depends on the following components being installed on you machine.

Java, VIPER requires the java virtual machine, it was developed and built with version 1.3.1 so any virtual machine after that should be ok.

NASM linker this should be on th PATH of your machine and the file should be name s nasm.exe.

The Gnu C Compiler for linking (GCC) on linux machines this usually comes as standard with the os, Windows users have the option of Cygwin or DJGPP.
