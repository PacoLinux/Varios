The compiler options must be set before any program can be compiled. VIPER will remember these options so if you only wish to use one set up then this will be set up once and left alone. The compiler options can be found in Options/Compiler Options. The Compiler Options can be set and reset at run time.

The -cpu flag should be set to the processor that you wish to create code for. If this is not the processor that is fitted to your computer clearly the Run Program menu items will not work . The -S and -V are optional. -S stops the linking stage of the compilation process and -V produces a verbose output. Please see the Vector Pascal Manual section for more details.

Windows...

...with DJGPP

The -f flag should be set. The -f text field should contain the text coff.

...with Cygwin

The -f flag should be set. The -f text field should contain the text elf.

Linux

The -f flag should be set and it's text field should contain the text elf.
